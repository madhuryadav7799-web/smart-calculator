## 🔌 WEBSOCKET EVENTS

Real-time communication via Socket.io:

```javascript
// Task Events
socket.emit('task:create', taskData)
socket.emit('task:update', taskId, updates)
socket.emit('task:start', taskId)
socket.emit('task:complete', taskId, result)

// Code Events
socket.emit('code:save', codeData)
socket.emit('code:execute', codeData)

// System Events
socket.on('system:stats', stats)      // Real-time updates
socket.on('logs:batch', logs)        // Log streaming
socket.on('stats:update', stats)     // Stats updates

// AI Events
socket.emit('ai:suggestion-request', context)
socket.on('ai:suggestions', suggestions)

// Settings
socket.emit('settings:update', settings)
```
