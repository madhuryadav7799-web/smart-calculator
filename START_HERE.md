## ⚡ Quick Start (5 Minutes)

The fastest way to get your APK:

```bash
# 1. Install EAS CLI (one time)
npm install -g eas-cli

# 2. Go to project directory
cd ~/.openclaw/workspace/apps/smart-calculator

# 3. Login (create free account)
eas login

# 4. Build APK
eas build --platform android --profile preview

# 5. Download when complete!
# Your APK is ready to install on any Android device
```

**Time:** 2-5 minutes build time + 5 min setup = **~10 minutes total**
