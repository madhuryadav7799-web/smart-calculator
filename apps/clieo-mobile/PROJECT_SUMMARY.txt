================================================================================
  NEXT STEPS - BUILD ANDROID APK
================================================================================

1. INSTALL DEPENDENCIES (if not done yet):
   $ cd /workspace/apps/clieo-mobile
   $ npm install

2. PREBUILD FOR ANDROID:
   $ npm run prebuild
   (Creates /android directory with native code)

3. BUILD APK:
   $ cd android
   $ ./gradlew assembleDebug
   (Takes 5-10 minutes on first build)

4. APK READY AT:
   android/app/build/outputs/apk/debug/app-debug.apk

5. INSTALL ON DEVICE:
   $ adb install android/app/build/outputs/apk/debug/app-debug.apk

6. OR SEND VIA TELEGRAM:
   Upload app-debug.apk to Telegram file chat
