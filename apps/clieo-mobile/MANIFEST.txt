BUILD INSTRUCTIONS
================================================================================

STEP 1: Install Dependencies
   $ cd /workspace/apps/clieo-mobile

   [Takes ~5-10 minutes]

STEP 3: Build APK

STEP 4: Output

STEP 5: Install
