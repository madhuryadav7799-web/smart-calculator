TROUBLESHOOTING QUICK REFERENCE
================================================================================

Problem: "eas login doesn't work"
Solution: npm cache clean --force && eas logout && eas login

Problem: "Java not found"
Solution: Install Java 11+ from https://adoptopenjdk.net/

Problem: "Android SDK not found"
Solution: Set ANDROID_HOME environment variable

Problem: "Build takes too long"
Solution: Use EAS Cloud instead (faster)

Problem: "APK won't install"
Solution: 
  - Check Android version (needs 10+)
  - Clear app data first
  - Try: adb install -r SmartCalculator-final.apk

Problem: "App crashes on startup"
Solution:
  - Grant permissions when prompted
  - Check device has 100MB+ RAM
  - View logs: adb logcat | grep ReactNativeJS

Full troubleshooting in: SMART_CALCULATOR_FINAL_APK_GUIDE.md
