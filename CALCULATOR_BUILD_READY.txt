QUICK START - BUILD APK IN 5 MINUTES
================================================================================

Step 1: Navigate to project
   cd /workspace/apps/smart-calculator

Step 2: Install dependencies
   npm install

Step 3: Install EAS CLI (first time only)
   npm install -g eas-cli

Step 4: Login to Expo (first time only)
   eas login

Step 5: Build APK
   eas build --platform android --profile preview

Step 6: Download APK when build completes (~60MB)

Step 7: Install on Android phone
   - Transfer APK to phone
   - Tap to install
   - Allow if prompted
   - Open from app drawer

That's it! Calculator is ready to use.
